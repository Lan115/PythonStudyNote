## Python字符串操作：查找、判断、修改

### 查找

#### find() 和 rfind()

```
text = "Hello, World! Welcome to Python."

# find() - 从左向右查找，返回第一个匹配的索引
index1 = text.find("World")  # 返回 7
index2 = text.find("Java")   # 返回 -1（未找到）

# rfind() - 从右向左查找，返回最后一个匹配的索引
index3 = text.rfind("o")     # 返回 26
```

#### index() 和 rindex()

```
# index() - 类似find()，但找不到时会抛出ValueError异常
index4 = text.index("World")  # 返回 7
# index5 = text.index("Java")   # ValueError: substring not found

# rindex() - 类似rfind()，但找不到时会抛出异常
index6 = text.rindex("o")     # 返回 26
```

#### count() - 统计出现次数

```
count_o = text.count("o")     # 返回 4
count_hello = text.count("Hello")  # 返回 1
```

### 判断

#### 开头和结尾判断

```
text = "Hello World"

# startswith() - 检查是否以指定字符串开头
starts_hello = text.startswith("Hello")  # True
starts_hello_lower = text.startswith("hello")  # False

# endswith() - 检查是否以指定字符串结尾
ends_world = text.endswith("World")  # True
ends_world_exclamation = text.endswith("World!")  # False
```

#### 字符类型判断

```
text1 = "Hello123"
text2 = "12345"
text3 = "HELLO"
text4 = "hello"
text5 = "   "
text6 = "Hello World"

# isalnum() - 是否只包含字母和数字
print(text1.isalnum())  # True
print(text6.isalnum())  # False（包含空格）

# isalpha() - 是否只包含字母
print("Hello".isalpha())  # True
print(text1.isalpha())    # False

# isdigit() - 是否只包含数字
print(text2.isdigit())  # True
print("123.45".isdigit())  # False

# islower() - 是否全部小写
print(text4.islower())  # True
print(text3.islower())  # False

# isupper() - 是否全部大写
print(text3.isupper())  # True
print(text4.isupper())  # False

# isspace() - 是否只包含空白字符
print(text5.isspace())  # True
print(text1.isspace())  # False
```

### 修改

#### 大小写转换

```
text = "Hello World"

# lower() - 转换为小写
lower_text = text.lower()  # "hello world"

# upper() - 转换为大写
upper_text = text.upper()  # "HELLO WORLD"

# capitalize() - 首字母大写
capital_text = text.capitalize()  # "Hello world"

# title() - 每个单词首字母大写
title_text = text.title()  # "Hello World"

# swapcase() - 大小写互换
swap_text = text.swapcase()  # "hELLO wORLD"
```

#### 替换操作

```
text = "I like apples, apples are tasty"

# replace() - 替换字符串
new_text = text.replace("apples", "oranges")  # "I like oranges, oranges are tasty"

# 指定替换次数
limited_replace = text.replace("apples", "oranges", 1)  # "I like oranges, apples are tasty"
```

#### 去除空白字符

```
text = "   Hello World   \n"

# strip() - 去除两端空白字符
stripped = text.strip()  # "Hello World"

# lstrip() - 去除左侧空白字符
left_stripped = text.lstrip()  # "Hello World   \n"

# rstrip() - 去除右侧空白字符
right_stripped = text.rstrip()  # "   Hello World"
```

#### 对齐和填充

```
text = "Hello"

# center() - 居中填充
centered = text.center(20, "*")  # "*******Hello********"

# ljust() - 左对齐填充
left_just = text.ljust(10, "-")  # "Hello-----"

# rjust() - 右对齐填充
right_just = text.rjust(10, "+")  # "+++++Hello"
```

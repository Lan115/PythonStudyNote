## 常用内置函数

### 数据类型转换函数

```
# 基本类型转换
int('123')      # 123
float('3.14')   # 3.14
str(100)        # '100'
bool(0)         # False
bool(1)         # True

# 集合类型转换
list('hello')           # ['h', 'e', 'l', 'l', 'o']
tuple([1, 2, 3])        # (1, 2, 3)
set([1, 2, 2, 3])       # {1, 2, 3}
dict([('a', 1), ('b', 2)])  # {'a': 1, 'b': 2}
```

### 数学相关函数

```
# 数值计算
abs(-5)         # 5 - 绝对值
round(3.14159, 2)  # 3.14 - 四舍五入
pow(2, 3)       # 8 - 幂运算
divmod(10, 3)   # (3, 1) - 商和余数

# 最大值最小值
max(1, 5, 3)    # 5
min([4, 2, 8])  # 2
sum([1, 2, 3])  # 6
```

### 迭代和序列函数

```
# 长度和排序
len('hello')        # 5
len([1, 2, 3])      # 3
sorted([3, 1, 2])   # [1, 2, 3] - 返回新列表
reversed([1, 2, 3]) # <list_reverseiterator> - 返回迭代器

# 枚举和zip
list(enumerate(['a', 'b', 'c']))  # [(0, 'a'), (1, 'b'), (2, 'c')]
list(zip([1, 2, 3], ['a', 'b', 'c']))  # [(1, 'a'), (2, 'b'), (3, 'c')]
```

## 拆包

### 基本拆包

```
# 元组拆包
a, b, c = (1, 2, 3)
print(a, b, c)  # 1 2 3

# 列表拆包
x, y, z = [4, 5, 6]
print(x, y, z)  # 4 5 6
```

### 星号(*)拆包 - 处理剩余元素

```
# 收集剩余元素
first, *middle, last = [1, 2, 3, 4, 5]
print(first)    # 1
print(middle)   # [2, 3, 4]
print(last)     # 5

# 忽略某些元素
a, _, c = (1, 2, 3)  # 忽略第二个元素
*_, last = [1, 2, 3, 4, 5]  # 只取最后一个元素
```

### 字典拆包

```
# 键的拆包
person = {'name': 'Alice', 'age': 25, 'city': 'Beijing'}
name, age, city = person  # 拆包得到键：'name', 'age', 'city'

# 键值对的拆包
for key, value in person.items():
    print(f"{key}: {value}")

# 双星号(**)字典拆包
def print_info(name, age, city):
    print(f"{name} is {age} years old, from {city}")

print_info(**person)  # 相当于 print_info(name='Alice', age=25, city='Beijing')
```

### 函数参数拆包

```
# 位置参数拆包
def add(a, b, c):
    return a + b + c

numbers = [1, 2, 3]
result = add(*numbers)  # 相当于 add(1, 2, 3)

# 关键字参数拆包
def greet(name, message="Hello"):
    return f"{message}, {name}!"

params = {'name': 'Bob', 'message': 'Hi'}
print(greet(**params))  # Hi, Bob!
```

### 嵌套拆包

```
# 嵌套结构拆包
data = [(1, 'a'), (2, 'b'), (3, 'c')]
for num, letter in data:
    print(f"Number: {num}, Letter: {letter}")

# 复杂嵌套
nested = [('Alice', 25, ['Python', 'Java']), ('Bob', 30, ['JavaScript'])]
for name, age, languages in nested:
    print(f"{name} knows {len(languages)} languages")
```

### 星号表达式的其他用法

```
# 合并列表
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined = [*list1, *list2]  # [1, 2, 3, 4, 5, 6]

# 合并字典
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}
merged = {**dict1, **dict2}  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# 如果有重复键，后面的会覆盖前面的
dict3 = {'a': 1, 'b': 2}
dict4 = {'b': 99, 'c': 3}
result = {**dict3, **dict4}  # {'a': 1, 'b': 99, 'c': 3}
```

## 内置函数与拆包的结合使用

### zip + 拆包

```
names = ['Alice', 'Bob', 'Charlie']
scores = [85, 92, 78]

# 同时遍历多个列表
for name, score in zip(names, scores):
    print(f"{name}: {score}")

# 转换为字典
score_dict = dict(zip(names, scores))  # {'Alice': 85, 'Bob': 92, 'Charlie': 78}
```

### enumerate + 拆包

```
fruits = ['apple', 'banana', 'orange']

for index, fruit in enumerate(fruits):
    print(f"Index {index}: {fruit}")

# 指定起始索引
for index, fruit in enumerate(fruits, start=1):
    print(f"#{index}: {fruit}")
```

### 函数返回多值 + 拆包

```
def get_user_info():
    return 'Alice', 25, 'alice@example.com'

name, age, email = get_user_info()  # 直接拆包返回值
```

### 处理可变参数函数

```
def print_args(*args, **kwargs):
    print("位置参数:", args)
    print("关键字参数:", kwargs)

print_args(1, 2, 3, name='Alice', age=25)
# 位置参数: (1, 2, 3)
# 关键字参数: {'name': 'Alice', 'age': 25}
```
